8051 basic usage using EdSim51

January 25, 2019 One minute read
8051

EdSim51

This section starts with directly working on EdSim51, and this means this section will do operation with assembly.

What is EdSim51 ?

Basically it simulates a complete embedded system, including LCD, LEDs, keypads, serial.

How to program it ?

  • directly on bare metal –> assembly :((

Well, its quite low-level, for that you need to manage all the registers, memory, control flow by yourself by writing assembly, which is really hard to read. But the trade off: you will know how a computer really work!!!

Let’s first try out light up its 7 segment display. To light up a 7 segment, we need to know which wire is connected to it, and which port is where we can send signal. Thus, let’s see the architecture of the segment.

So port P1 (address 90H) is connected to this 7 segment display.

ORG 0H   
MOV 90H, #24H
END
  • use SDCC compiler to translate your C code (human readable thankfully~) to assembly code
